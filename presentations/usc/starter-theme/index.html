What is it?

It is (becoming) a stripped down version of Underscores (_s).

It will also grow with enhancements to our group added as we continue.

Goals

Primary goal will to have a base theme that we can take a snapshot of for development.

  • Functions and variables folder as main editing sections
  • Well formed HTML outline/algorithm for site indexing and search (SEO)
  • Faster deployments

Semantics

The framework thus far has a good semantic algorithmic outline.

Theme Starter has the proper use of article, section, asides, div's, etc*

* If not, note the change and it will be made

Design

A majority of our sites are agnostic with regards to document structure and design

Having a styleguide will allow us to write a visually harmonious site:

  • Incremental heading sizes
  • Same treatment of elements across site
  • Did I mention color palettes?

Take Aways

  • Base theme to take snapshot and start developing
  • Control of the styleguide and components in a few files (majority)
    • functions.php
    • styleguide/_typography.scss
    • styleguide/_grid.scss
    • variables/_colors.scss
    • variables/_variables.scss
  • Faster deployments
  • Allows future enhancements
